Athens Bus App.
(1.0.0-beta)

--

Αντικείμενο αυτής της mobile εφαρμογής είναι η πληροφόρηση για τις αναμενόμενες αφίξεις στις στάσεις των λεωφορείων και των τρόλεϊ της Αθήνας.
The subject of this mobile app is to inform about the expected arrivals at the bus stations of Athens.

Η ανάπτυξη της εφαρμογής έγινε από τον GiannisClipper, ο κώδικας είναι γραμμένος σε React Native και τα δεδομένα προέρχονται από το OASA telematics API.
App developed by GiannisClipper, code written in React Native and data powered by OASA telematics API.

Πρόκειται για μια ανεξάρτητη πρωτοβουλία, που δεν σχετίζεται με τον Oργανισμό Aστικών Συγκοινωνιών Αθηνών, με σκοπό την εξάσκηση και την παρουσίαση προγραμματιστικών ικανοτήτων.
It is an indypendent initiutive, not associated with transportation organization of Athens (OASA), for the purpose of practicing and demonstrate programming skills.

--

Στο κυρίως μενού της εφαρμογής υπάρχουν τρεις επιλογές:
In the main menu of the app there are three choices:

Η επιλογή 'HOME' εμφανίζει την αρχική οθόνη με ένα μικρό εισαγωγικό κείμενο. 
The 'HOME' choice displays the home page with a small introductory text. 

Η επιλογή 'LINES' εμφανίζει λίστες με τις λεωφορειακές γραμμές, τις δραδρομές ανά γραμμή, τις στάσεις ανά διαδρομή και τις αναμενόμενες αφίξεις ανά στάση.
The 'LINES' choice displays lists with the bus lines, the routes per line, the stops per route and the expected arrivals per stop. 

Η επιλογή 'MySTOPS' εμφανίζει λίστες με τις αγαπημένες στάσεις που έχει ορίσει ο χρήστης και τις αναμενόμενες αφίξεις για κάθε μία από αυτές τις στάσεις.
The choice 'MySTOPS' displays a list with the favourite stops defined by the user and the expected arrivals per favourite stop.

--

Η επιλογή 'LINES' εμφανίζει τις λεωφορειακές γραμμές ομαδοποιημένες σύμφωνα με το αρχικό ψηφίο του αριθμού τους ή σύμφωνα με την αναζήτηση που μπορεί να κάνει ο χρήστης πληκτρολογώντας μέρος του αριθμού ή της περιγραφής τους.
The choice 'LINES' displays the bus lines displayed in groups according to the first digit of their line ID or according to the search user may commit by typing a part of the line IDs or descriptions. 

Για κάθε γραμμή εμφανίζεται ο αριθμός της και η περιγραφή της, το μπλε χρώμα στον αριθμό δηλώνει τις γραμμές των λεωφορείων ενώ το κίτρινο τις γραμμές των τρόλεϊ.
For each line is displayed the ID and the description, the blue color around the ID declares the bus lines while the yellow the trolley lines. 
--

Στη συνέχεια με βάση τις επιλογές του χρήστη, μπορούν να εμφανιστούν οι διαδρομές για κάθε γραμμή, οι στάσεις για κάθε διαδρομή και οι αναμενόμενες αφίξεις για κάθε στάση.
Subsequently responding to the user selections, may be displayed the routes of each line, the stops of each route and the expected arrivals of each stop. 
--

Στη λίστα με τις αναμενόμενες αφίξεις για κάποια στάση υπάρχει ένα ειδικότερο μενού.
In the list with expected arrivals at a bus stop there is a more specific menu.

Η επιλογή 'CLOSE' οδηγεί στην προηγούμενη οθόνη ενώ η επιλογή 'MySTOP' χρησιμοποιείται για τον χαρακτηρισμό ή τον αποχαρακτηρισμό μιας στάσης ως αγαπημένης.
The 'CLOSE' choice navigates to the previous screen while the 'MySTOP' choice is used to select or deselect the stop as favourite.

Επιπλέον σε αυτή τη λίστα, οι αφίξεις και οι χρόνοι αναμονής ανανεώνονται αυτόματα κάθε 20 δευτερόλεπτα.
Furthermore in this list, the arrivals and the delay times are getting updated automatically every 20 seconds.
--

Η επιλογή 'MySTOPS' προσφέρει άμεση πρόσβαση στις στάσεις που έχουν επιλεγεί ως αγαπημένες από τον χρήστη καθώς και στις λίστες με τις αναμενόμενες αφίξεις για κάθε μία από αυτές.
The 'MySTOPS' choice offers instand access to the stops that have been defined as favourites by the user as well as to the lists with the expected arrivals for each of them.
